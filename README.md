## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/tic-tac-toe-MVC.git
```

Go to the project directory

```bash
  cd tic-tac-toe-MVC
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
